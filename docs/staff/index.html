Moderation Commands

Command Usage Description
warn !!warn <user> <reason> DM's the user the warning provided by the punisher
kick !!kick <user> <reason> Removes a user from the server for misconduct.
ban !!ban <user> <reason> Banishes a user from the server for misconduct.
unban !!unban <user> <reason> Un-banishes a user from the server.
mute !!mute <user> <time (s/m/h/d/m/y)> <reason> Prevents the user from speaking for the amount of time given.
unmute !!unmute <user> <reason> Grants a user permission to speak.
react !!react <emoji/emoji id> Make the bot react to the previous message. (Custom Emjoi need IDs)

Management Commands

To create a ticket, react to the message in #tickets on discord.

Command Usage Description
addtrigger !!addtrigger <type> <trigger> Adds a message trigger
remtrigger !!remtrigger <type> <trigger> Removes a message trigger
announce !!announce "<Title>" "<Description>" "<Type>" "<Ping True (D)/Fase>" Announces a mesage. (ALL ARGUMENTS NEED TO BE SURROUNDED BY "")
embed !!embed "<Title>" "<Description>" Creates an embed. (ALL ARGUMENTS NEED TO BE SURROUNDED BY "")
giveaway !!giveaway <start|end> <amount of winners|id> <prize> Ends/starts a giveaway
whitelist !!whitelist <add|remove> <meme> <subreddit> Whitelists/unwhitelists a subreddit for !!meme

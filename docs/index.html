Getting Started Last updated: 2020-04-12

This will cover the basics of MiniJoshua.



The Basics

Run the command, !!prefix to see the guild's current prefix.

Commands

MiniJoshua's Commands.

(): Required

[] : Optional

Basic Commands

Command Usage Description
docs !!docs [Plugin#Document] Shows you the documents of JoshuaLovesCode's plugins.
donate !!donate Shows you a link where you can donate.
help !!help [command] Displays the commands accesible to you.
info !!info Displays the bots current info.
npm !!npm <package> Displays information about a certain node-package.
partners !!partners Shows the partners of JoshuaLovesCode
patrons !!patrons Shows JoshuaLovesCode's patreons
rules !!rules Displays the current rules for JoshuaLovesCode's Discord Server.
spigot !!spigot <plugin/user> <name/id> Gets a plugin or author's information from SpigotMC

Ticket Commands

To create a ticket, react to the message in #tickets on discord.

Command Usage Description
add !!add <username> Adds a user to the ticket
remove !!remove <username> Removes a user from the ticket
close !!close Closes the ticket, with a confirmation

Fun Commands

Command Usage Description
8ball !!8ball <question> Shake a magic 8 ball to see your luck.
coinflip !!coinflip Toss a coin that lands on heads or tails.
dice !!dice [amount of dice] Rolls some dice. By default, it rolls 1x 6 sided dice.
level !!level [user] Display's your XP and level. Get more by chatting.
meme !!meme [subreddit] Shows you top quality memes, even from a subreddit of your desire.
rockpaperscissors !!rps Play a game of rock paper scissors against the bot.

Moderation Commands

If you are a moderator on JoshuaLovesCode, check the pinned messages in #mod-chat for a link to the moderation section.
